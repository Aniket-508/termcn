## API Reference

### Badge

| Prop          | Type           | Default     |
| ------------- | -------------- | ----------- |
| `children`    | `string`       | `required`  |
| `variant`     | `BadgeVariant` | `"default"` |
| `color`       | `string`       | `undefined` |
| `bold`        | `boolean`      | `false`     |
| `bordered`    | `boolean`      | `true`      |
| `borderStyle` | `BorderStyle`  | `"round"`   |
| `paddingX`    | `number`       | `1`         |
