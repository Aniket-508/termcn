## Props

| Prop                  | Type                                                                                            | Default     | Description                                                              |
| --------------------- | ----------------------------------------------------------------------------------------------- | ----------- | ------------------------------------------------------------------------ |
| `children`            | `string`                                                                                        | `required`  | -                                                                        |
| `language`            | `string`                                                                                        | `undefined` | -                                                                        |
| `inline`              | `boolean`                                                                                       | `false`     | -                                                                        |
| `borderStyle`         | `\| "single" \| "double" \| "round" \| "bold" \| "singleDouble" \| "doubleSingle" \| "classic"` | `"single"`  | Border style. Default: 'single'                                          |
| `showLineNumbers`     | `boolean`                                                                                       | `true`      | Whether to show line numbers. Default: true                              |
| `lineNumberSeparator` | `string`                                                                                        | `"│ "`      | Line number separator string. Default: '│ '                              |
| `keywordColor`        | `string`                                                                                        | `undefined` | Override color for keyword tokens. Default: theme.colors.accent          |
| `stringColor`         | `string`                                                                                        | `undefined` | Override color for string tokens. Default: theme.colors.success          |
| `numberColor`         | `string`                                                                                        | `undefined` | Override color for number tokens. Default: theme.colors.warning          |
| `commentColor`        | `string`                                                                                        | `undefined` | Override color for comment tokens. Default: theme.colors.mutedForeground |
| `operatorColor`       | `string`                                                                                        | `undefined` | Override color for operator tokens. Default: theme.colors.info           |
| `plainColor`          | `string`                                                                                        | `undefined` | Override color for plain tokens. Default: theme.colors.foreground        |
