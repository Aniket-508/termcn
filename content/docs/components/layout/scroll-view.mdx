## Usage

```tsx
import { ScrollView } from "@/components/ui/scroll-view";
```

```tsx
<ScrollView height={5}>
  <Text>Line 1</Text>
  <Text>Line 2</Text>
  <Text>Line 3</Text>
</ScrollView>
```

## API Reference

### ScrollView

| Prop             | Type        | Default  |
| ---------------- | ----------- | -------- |
| `height`         | `number`    | required |
| `children`       | `ReactNode` | required |
| `showScrollbar`  | `boolean`   | `true`   |
| `scrollbarColor` | `string`    | `"gray"` |
