
## Props

| Prop              | Type                      | Default    | Description |
| ----------------- | ------------------------- | ---------- | ----------- |
| `models`          | `ModelOption[]`           | `required` | -           |
| `selected`        | `string`                  | `required` | -           |
| `onSelect`        | `(id: string) =&gt; void` | `required` | -           |
| `showContext`     | `boolean`                 | `true`     | -           |
| `showProvider`    | `boolean`                 | `true`     | -           |
| `groupByProvider` | `boolean`                 | `false`    | -           |
