## Installation

```bash
npx shadcn@latest add https://termcn.vercel.app/r/thinking-block.json
```

## Usage

```tsx
import { ThinkingBlock } from "@/components/ui/thinking-block";

export function Example() {
  return (
    <ThinkingBlock
      content="The user wants to refactor the function. I should split it into smaller helpers and add proper error handling..."
      tokenCount={1284}
      duration={3200}
    />
  );
}
```

## Props

| Prop               | Type      | Default       | Description                                                     |
| ------------------ | --------- | ------------- | --------------------------------------------------------------- |
| `content`          | `string`  | `required`    | The reasoning/thinking text to display inside the block.        |
| `streaming`        | `boolean` | `false`       | Show a "Thinking..." header while the model is still reasoning. |
| `defaultCollapsed` | `boolean` | `true`        | Whether the block starts collapsed; toggle with Enter or Space. |
| `label`            | `string`  | `"Reasoning"` | Header label shown when not streaming.                          |
| `tokenCount`       | `number`  | `undefined`   | Number of thinking tokens used, displayed in the header.        |
| `duration`         | `number`  | `undefined`   | Thinking duration in milliseconds, shown as seconds in header.  |

## Examples

### Collapsed by default

```tsx
<ThinkingBlock
  content="Let me analyze the error stack trace to find the root cause..."
  tokenCount={832}
  duration={2100}
/>
```

### Expanded on render

```tsx
<ThinkingBlock
  content="I need to consider edge cases: empty arrays, null values, and circular references."
  defaultCollapsed={false}
  label="Chain of thought"
/>
```

### Currently streaming

```tsx
<ThinkingBlock
  content="Working through the problem step by step..."
  streaming
  defaultCollapsed={false}
/>
```
