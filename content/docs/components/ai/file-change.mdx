## Installation

```bash
npx shadcn@latest add https://termcn.vercel.app/r/file-change.json
```

## Usage

```tsx
import { FileChange } from "@/components/ui/file-change";

export function Example() {
  const demoProps = {
    // See the live preview for a working example.
  };

  return <FileChange {...demoProps} />;
}
```

| Prop          | Type                        | Default     | Description |
| ------------- | --------------------------- | ----------- | ----------- |

| `onAccept`    | `(path: string) =&gt; void` | `undefined` | -           |
| `onReject`    | `(path: string) =&gt; void` | `undefined` | -           |
| `onAcceptAll` | `() =&gt; void`             | `undefined` | -           |
