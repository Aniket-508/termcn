```bash
npx shadcn@latest add https://termcn.vercel.app/r/streaming-text.json
```

## Usage

```tsx

  const demoProps = {
    // See the live preview for a working example.
  };
```

## Props

| Prop          | Type                            | Default     | Description                                                  |
| ------------- | ------------------------------- | ----------- | ------------------------------------------------------------ |
| `text`        | `string`                        | `undefined` | Current accumulated string to display (controlled mode)      |
| `stream`      | `AsyncIterable&lt;string&gt;`   | `undefined` | AsyncIterable&lt;string&gt; source; manages state internally |
| `cursor`      | `boolean`                       | `true`      | Show blinking ▌ cursor at end while streaming                |
| `animate`     | `boolean`                       | `false`     | Simulate typing animation for pre-buffered text              |
| `speed`       | `number`                        | `30`        | Typing animation speed in ms per character (default 30)      |
| `onComplete`  | `(fullText: string) =&gt; void` | `undefined` | Called with full text when streaming/animation completes     |
| `cursorColor` | `string`                        | `undefined` | Color for the cursor                                         |
