## API Reference

### Spinner

| Prop     | Type                                                                                                                       | Default     |
| -------- | -------------------------------------------------------------------------------------------------------------------------- | ----------- |
| `type`   | `"dots" \| "line" \| "star" \| "clock" \| "bounce" \| "bar" \| "arc" \| "arrow" \| "toggle" \| "box" \| "pipe" \| "earth"` | `"dots"`    |
| `label`  | `string`                                                                                                                   | `undefined` |
| `color`  | `string`                                                                                                                   | `undefined` |
| `fps`    | `number`                                                                                                                   | `12`        |
| `frames` | `string[]`                                                                                                                 | `undefined` |
