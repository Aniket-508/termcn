## Usage

```tsx
import { MultiSelect } from "@/components/ui/multi-select";
```

```tsx
<MultiSelect
  options={[
    { value: "react", label: "React" },
    { value: "vue", label: "Vue" },
    { value: "svelte", label: "Svelte" },
  ]}
  onChange={(values) => {}}
/>
```

## API Reference

### MultiSelect

| Prop        | Type                     | Default     |
| ----------- | ------------------------ | ----------- |
| `options`   | `MultiSelectOption<T>[]` | `required`  |
| `value`     | `T[]`                    | `undefined` |
| `onChange`  | `(values: T[]) => void`  | `undefined` |
| `onSubmit`  | `(values: T[]) => void`  | `undefined` |
| `cursor`    | `string`                 | `"›"`       |
| `checkmark` | `string`                 | `"◉"`       |
| `height`    | `number`                 | `undefined` |
