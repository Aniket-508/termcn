## API Reference

### QRCode

| Prop    | Type                   | Default     |
| ------- | ---------------------- | ----------- |
| `value` | `string`               | `required`  |
| `size`  | `"sm" \| "md" \| "lg"` | `"md"`      |
| `color` | `string`               | `undefined` |
| `label` | `string`               | `undefined` |
