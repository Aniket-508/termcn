
| Prop            | Type         | Default     | Description |
| --------------- | ------------ | ----------- | ----------- |

| `height`        | `number`     | `10`        | -           |
| `showTimestamp` | `boolean`    | `true`      | -           |
| `filter`        | `string`     | `undefined` | -           |
| `follow`        | `boolean`    | `undefined` | -           |
