## Props

| Prop          | Type             | Default     | Description |
| ------------- | ---------------- | ----------- | ----------- |
| `format`      | `"12h" \| "24h"` | `"24h"`     | -           |

| `showDate`    | `boolean`        | `false`     | -           |
| `timezone`    | `string`         | `undefined` | -           |
| `color`       | `string`         | `undefined` | -           |
| `size`        | `"sm" \| "lg"`   | `"sm"`      | -           |
