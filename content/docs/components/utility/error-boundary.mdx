## API Reference

### ErrorBoundary

| Prop       | Type                                                       | Default     |
| ---------- | ---------------------------------------------------------- | ----------- |
| `children` | `ReactNode`                                                | `required`  |
| `fallback` | `ReactNode`                                                | `undefined` |
| `onError`  | `(error: Error, info: { componentStack: string }) => void` | `undefined` |
| `title`    | `string`                                                   | `"Error"`   |
