
## Props

| Prop        | Type      | Default     | Description                                   |
| ----------- | --------- | ----------- | --------------------------------------------- |
| `data`      | `unknown` | `required`  | -                                             |
| `indent`    | `number`  | `2`         | Initial indent spaces. Default: 2             |
| `collapsed` | `boolean` | `false`     | Whether nodes start collapsed. Default: false |
| `label`     | `string`  | `undefined` | -                                             |
