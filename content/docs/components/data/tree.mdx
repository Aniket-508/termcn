## Props

| Prop              | Type                          | Default     | Description                                                         |
| ----------------- | ----------------------------- | ----------- | ------------------------------------------------------------------- |
| `nodes`           | `TreeNode[]`                  | `required`  | -                                                                   |
| `onSelect`        | `(node: TreeNode) =&gt; void` | `undefined` | -                                                                   |
| `defaultExpanded` | `string[]`                    | `[]`        | -                                                                   |
| `expandedIcon`    | `string`                      | `"▼"`       | Icon shown for expanded branch nodes. Default: '▼'                  |
| `collapsedIcon`   | `string`                      | `"▶"`       | Icon shown for collapsed branch nodes. Default: '▶'                 |
| `leafIcon`        | `string`                      | `"•"`       | Default icon for leaf nodes (overridden by node.icon). Default: '•' |
