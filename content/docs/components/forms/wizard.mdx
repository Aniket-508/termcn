## Installation

```bash

```

## Usage

```tsx
import { Wizard } from "@/components/ui/wizard";

export function Example() {
  const demoProps = {
    // See the live preview for a working example.
  };

  return <Wizard {...demoProps} />;
}
```

| `onCancel`     | `() =&gt; void`                         | `undefined` | -           |
| `showProgress` | `boolean`                               | `true`      | -           |
