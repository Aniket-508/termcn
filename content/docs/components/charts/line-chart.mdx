## Installation

```bash
npx shadcn@latest add https://termcn.vercel.app/r/line-chart.json
```

## Usage

```tsx
import { LineChart } from "@/components/ui/line-chart";

export function Example() {
  const demoProps = {
    // See the live preview for a working example.
  };

  return <LineChart {...demoProps} />;
}
```

| `data`     | `LineChartDataPoint[]` | `required`  | -           |

| `height`   | `number`               | `10`        | -           |
| `title`    | `string`               | `undefined` | -           |
| `color`    | `string`               | `undefined` | -           |
| `showAxes` | `boolean`              | `true`      | -           |
