## API Reference

### Gauge

| Prop    | Type                   | Default     |
| ------- | ---------------------- | ----------- |
| `value` | `number`               | `required`  |
| `min`   | `number`               | `0`         |
| `max`   | `number`               | `100`       |
| `label` | `string`               | `undefined` |
| `color` | `string`               | `undefined` |
| `size`  | `"sm" \| "md" \| "lg"` | `"md"`      |
