## Usage

```tsx
import { RadioGroup } from "@/components/ui/radio-group";
```

```tsx
<RadioGroup
  options={[
    { value: "sm", label: "Small" },
    { value: "md", label: "Medium" },
    { value: "lg", label: "Large" },
  ]}
  onChange={(value) => {}}
/>
```

## API Reference

### RadioGroup

| Prop       | Type                 | Default     |
| ---------- | -------------------- | ----------- |
| `options`  | `RadioOption<T>[]`   | `required`  |
| `value`    | `T`                  | `undefined` |
| `onChange` | `(value: T) => void` | `undefined` |
| `name`     | `string`             | `undefined` |
| `cursor`   | `string`             | `"›"`       |
