```

```

## API Reference

### Alert

| Prop          | Type           | Default     |
| ------------- | -------------- | ----------- |
| `variant`     | `AlertVariant` | `"info"`    |
| `title`       | `string`       | `undefined` |
| `children`    | `ReactNode`    | `undefined` |
| `icon`        | `string`       | `undefined` |
| `bordered`    | `boolean`      | `true`      |
| `borderStyle` | `BorderStyle`  | `undefined` |
| `color`       | `string`       | `undefined` |
| `paddingX`    | `number`       | `1`         |
| `paddingY`    | `number`       | `0`         |
