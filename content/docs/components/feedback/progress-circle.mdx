## API Reference

### ProgressCircle

| Prop          | Type                   | Default     |
| ------------- | ---------------------- | ----------- |
| `value`       | `number`               | `required`  |
| `size`        | `"sm" \| "md" \| "lg"` | `"sm"`      |
| `color`       | `string`               | `undefined` |
| `label`       | `string`               | `undefined` |
| `showPercent` | `boolean`              | `false`     |
