## API Reference

### Banner

| Prop          | Type                                                       | Default     |
| ------------- | ---------------------------------------------------------- | ----------- |
| `children`    | `ReactNode`                                                | `required`  |
| `variant`     | `"info" \| "warning" \| "error" \| "success" \| "neutral"` | `"info"`    |
| `icon`        | `string`                                                   | `undefined` |
| `title`       | `string`                                                   | `undefined` |
| `dismissible` | `boolean`                                                  | `false`     |
| `onDismiss`   | `() => void`                                               | `undefined` |
| `color`       | `string`                                                   | `undefined` |
| `accentChar`  | `string`                                                   | `"┃"`       |
| `gap`         | `number`                                                   | `1`         |
