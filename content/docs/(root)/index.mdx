description: "termcn ships a shadcn-style registry for 113 Ink terminal components and bundled themes."
---

**termcn** publishes a large Ink component registry that installs through the shadcn CLI while keeping the source code in your app.

## What You Get

- 113 terminal UI components grouped by category
- Bundled terminal theming with live browser previews
- Registry items for UI components, hooks, and theme files
- Static docs pages with client-side xterm.js previews

## Install Any Component

```bash
npx shadcn@latest add https://termcn.vercel.app/r/badge.json
```

## Themes

The docs preview every component inside a browser terminal and let you switch between the bundled terminal themes live.
